# HTML Exports

Pre-rendered interactive visualizations generated from `vital_load.ipynb`.  
ECG_II (500Hz) is excluded due to file size constraints.

| File | Description |
|------|-------------|
| [waveform.html](https://SEOL8.github.io/ICU-Vital-Signs-Processing-Pipeline/html/waveform.html) | ABP, CVP, PLETH waveforms (125Hz) |
| [numeric.html](https://SEOL8.github.io/ICU-Vital-Signs-Processing-Pipeline/html/numeric.html) | HR, BP, SpO2, RR numeric parameters (1Hz) |

> Data is embedded at export time. These files do not contain patient-identifiable information.
